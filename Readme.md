## Installation

To install this package, simply run:

```bash
npm i
```

## Usage

Make sure you are using Node.js version 18.20 or higher.

After installation, you can run the following scripts to run each problem:

### Problem 1
```bash
npm run problem-1
```

### Problem 2
```bash
npm run problem-2

```

### Problem 3
```bash
npm run problem-3
```

### Problem 4
```bash
npm run problem-4
```
